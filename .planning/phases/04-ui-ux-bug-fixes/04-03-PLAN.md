<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| App output → OS clipboard | User-initiated copy writes potentially sensitive local certificate/key material to system clipboard. |
| Long local output → scrollable UI | Large strings are rendered into local GPUI views. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-07 | Information Disclosure | Clipboard copy buttons | mitigate | Copy buttons are explicit, adjacent to the copied target, labeled `复制`, and show `已复制`; no automatic copy occurs. |
| T-04-08 | Denial of Service | Long output rendering | accept | Local desktop rendering only; horizontal/vertical scrolling contains large text without blocking layout. |
| T-04-09 | Spoofing | Copy status banner | mitigate | Banner text is set only after `write_to_clipboard` call in copy helper and uses exact Chinese success string `已复制`. |
</threat_model>
